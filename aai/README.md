# Aprendizaje automático I

En esta carpeta se incluyen recursos de apoyo educativo para la asignatura de Aprendizaje automático I del MUICD de la UNED, con código 31110037.

Más información en la [ficha web oficial](https://www.uned.es/universidad/inicio/estudios/masteres/master-universitario-en-ingenieria-y-ciencia-de-datos/asignaturas.html?codAsignatura=31110037&codTitulacion=311001&idContenido=1).

## Contenido

La carpeta se organiza en la siguiente estructura:

- **[`apuntes`](apuntes/)**: Incluye apuntes originales.
- **[`exam`](exam/)**: Incluye soluciones y enunciados inspirados en exámenes anteriores.
- **[`resumen`](resumen/)**: Incluye resúmenes y síntesis del temario.

## Evaluación

La asignatura se evalúa a partir de las siguientes actividades:

| Actividad | % |
| - | - |
| Actividad 1 | 15 |
| Actividad 1 | 15 |
| Actividad 3 | 20 |
| Exámen | 50 |

Para aprobar la asignatura se requiere una nota final del 50% del máximo y superar el mínimo para aprobar las PECs evaluadas.

### PECs

Para aprobar la asignatura se requiere obtenga un mínimo de un 40% de la nota máxima de las PECs.

### Exámen

La **puntuación mínima del exámen** para aprobar la asignatura depende de lo que hayas sacado en las PECs. Se calcula como

$$ x = \frac{125 - y}{150} \Leftarrow 6x + 4\frac{y}{100} = 5$$

donde $y$ es el porcentaje de puntuación obtenida sobre la máxima posible en los PECs.

$$ x = \frac{25 - 2y}{30} \Leftarrow 6x + 4\frac{y}{10} = 5$$

donde $y$ es la puntuación sobre 10 obtenida en los PECs.

El exámen consta de dos partes:

- Test
  - 4 preguntas tipo test con 4 opciones posibles
  - Puntuación máxima: 5 puntos
  - Pregunta correcta: +0.5
  - Pregunta incorrecta: -0.1
- Desarrollo
  - Puntuación máxima: 5 puntos
  - 1 ejercicio de desarrollo

La puntuación de cada parte compensa y no es necesario obtener una nota mínima en cada apartado.
