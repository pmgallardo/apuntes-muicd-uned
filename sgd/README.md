# Seguridad de la gestión de datos

En esta carpeta se incluyen recursos de apoyo educativo para la asignatura de Seguridad de la gestión de datos (SGD) del MUICD de la UNED.

Para más información, consulta la [ficha oficial](https://www.uned.es/universidad/inicio/estudios/masteres/master-universitario-en-ingenieria-y-ciencia-de-datos/asignaturas.html?codAsignatura=31110126&codTitulacion=311001&idContenido=1).

## Prerrequisitos

La [guía de la asignatura](https://www.uned.es/universidad/inicio/estudios/masteres/master-universitario-en-ingenieria-y-ciencia-de-datos/asignaturas.html?codAsignatura=31110126&codTitulacion=311001&idContenido=2) no indica que se requiera haber o estar cursando otras asignaturas como prerrequisito.

No obstante, algunos alumnos aconsejan haber cursado la siguiente asignatura, si bien no es estrictamente necesario:

- [ICPDM](../icpdm/): la práctica 1 consiste en aplicar un gateway de seguridad perimetral Apache Knox a una infraestructura big data compatible con Apache Hadoop.

## Bibliografía

La bibliografía complementaria proporcionada en la ficha de la asignatura parece desactualizada, pues contiene títulos descatalogados y los más recientes son del 2019, cuando se inauguró el máster.

- Básica
  - Material original proporcionado para la asignatura.

- Complementaria
  - AVEN, Jeffery. PART IV: MANAGING SPARK, BOOK: SAMS TEACH YOURSELF APACHE SPARK IN 24 HOURS. Agosto 2016. SAMS. ISBN(13): 9780134445786
  - ALAPATI, Sam R. Expert Hadoop Administration. Addison-Wesley Professional. Addison-Wesley Object Technology Series. Diciembre 2016. ISBN(13): 9780134598147
  - GUPTA, Saurabh, GIRI, Venkata. Practical Enterprise Data Lake Insights: Handle Data-drive Challenges in an Enterprise Big Data Lake. APRESS, junio 2018. ISBN(13): 9781484235225
