## VD.EX.2024J2.1

### Enunciado VD.EX.2024J2.1

Se pretende estudiar cómo se utilizan los vídeos de un curso concreto de la UNED para evaluar su utilidad y su nivel de complejidad. Para ello se ha incorporado a la plataforma un sistema que registra: cuántos estudiantes acceden a cada vídeo, cuántas visualizaciones realiza cada estudiante, la franja horaria de acceso (mañana, tarde o noche) y el tiempo de permanencia en minutos. El objetivo es construir un conjunto de visualizaciones a partir de estos datos.

a. Describe las etapas necesarias para crear estas visualizaciones siguiendo un proceso estructurado de análisis de la información.

b. Indica qué preguntas clave deberían guiar el análisis y qué tipo de respuestas se espera obtener.

c. Señala qué roles, de los estudiados en la asignatura, participarían en este proceso de visualización.

### Solución VD.EX.2024J2.1

**a. Pasos según el proceso de análisis de la información**:

- Definición del objetivo: determinar utilidad (frecuencia de uso) y complejidad (tiempo de permanencia y número de revisitas).
- Recolección y estructuración de los datos por vídeo y por estudiante.
- Preprocesado: limpieza de registros, tratamiento de valores inconsistentes y creación de métricas derivadas.
- Cálculo de indicadores como tiempo medio de visualización:

$$
\text{Tiempo medio} =
\frac{\text{Minutos totales visualizados}}
{\text{Número total de visitas}}
$$

- Análisis de patrones por vídeo y franja horaria.
- Selección de representaciones adecuadas según el tipo de variable.
- Evaluación y mejora de la visualización.

**b. Preguntas clave**:

- ¿Qué vídeos son más utilizados?
- ¿Qué vídeos presentan mayor tiempo medio de permanencia?
- ¿Existen diferencias por franja horaria?
- ¿Existe relación entre número de visitas y tiempo de permanencia?

Estas preguntas permiten analizar utilidad y complejidad.

**c. Roles que intervienen**:

- Iniciador.
- Científico de datos.
- Informático.
- Diseñador.
- Comunicador.

## VD.EX.2024J2.2

### Enunciado VD.EX.2024J2.2

Considera las siguientes variables relacionadas con la monitorización de un edificio inteligente y responde a las cuestiones planteadas.

a. Clasifica cada variable como cualitativa (nominal u ordinal) o cuantitativa, justificando tu decisión. Si procede, establece relaciones jerárquicas entre variables del mismo tipo.

- Tipo de dispositivo: termostatos, sensores de luz y humedad, cámaras de presencia o de seguridad (estas últimas pueden ser estáticas o de 360°).
- Ubicación del dispositivo: planta, oficina, sala de reuniones, pasillo, cocina o baño.
- Estado del dispositivo: activo, inactivo o en mantenimiento.
- Consumo energético: medido en kWh.
- Temperatura: expresada en grados Celsius.

b. Propón la estructura de datos que consideres más adecuada para almacenar esta información, explicando sus ventajas e inconvenientes.

c. Algunos termostatos figuran como activos pero generan valores de temperatura no numéricos (NaN). Además, los sensores de luz y humedad, así como las cámaras, siempre presentan NaN en la temperatura. Si se desea analizar la temperatura del edificio, ¿qué técnicas de tratamiento de datos aplicarías? Incluye código ilustrativo adaptado a la estructura de datos elegida.

### Solución VD.EX.2024J2.2

**a. Clasificación de variables**:

- Tipo de dispositivo: cualitativa nominal.
- Ubicación del dispositivo: cualitativa nominal con posible jerarquía (planta → estancia).
- Estado del dispositivo: cualitativa nominal.
- Consumo de energía (kWh): cuantitativa continua.
- Temperatura (°C): cuantitativa continua.

**b. Estructura de datos adecuada**:

Estructura tipo tabla relacional (DataFrame).

Ventajas:

- Permite almacenar variables heterogéneas.
- Facilita filtrado y agregación.
- Compatible con herramientas de análisis.

Inconvenientes:

- No representa explícitamente jerarquías espaciales.
- Puede requerir claves externas en sistemas complejos.

**c. Técnicas para filtrar temperatura**:

Filtrar solo termostatos y eliminar valores NaN.

```python
import pandas as pd

df_filtrado = df[df["Tipo de dispositivo"] == "termostato"]
df_filtrado = df_filtrado.dropna(subset=["Temperatura"])
```

Alternativamente, imputacion:

```python
df_filtrado["Temperatura"] = df_filtrado["Temperatura"].fillna(
    df_filtrado["Temperatura"].mean()
)
```

## VD.EX.2024J2.3

### Enunciado VD.EX.2024J2.3

Evalúa la idoneidad de distintas características visuales para representar variables cuantitativas, ordinales y categóricas con más de siete posibles valores.

a. En cada celda de la tabla, indica una de las siguientes valoraciones: ideal (+), capacidad limitada (—) o no adecuada (X).

b. Justifica específicamente tu valoración para las características “color — tonalidad” y “color — intensidad”.

| Característica visual | Subtipo                | Variable cuantitativa | Variable ordinal | Variable categórica |
| --------------------- | ---------------------- | --------------------- | ---------------- | ------------------- |
| Color                 | Tonalidad (matiz, hue) |                       |                  |                     |
| Color                 | Intensidad             |                       |                  |                     |
| Color                 | Saturación             |                       |                  |                     |
| Forma                 | Orientación            |                       |                  |                     |
| Forma                 | Forma                  |                       |                  |                     |
| Forma                 | Cantidad               |                       |                  |                     |
| Posición espacial     | Posición 2D            |                       |                  |                     |
| Movimiento            | Desplazamiento         |                       |                  |                     |

### Solución VD.EX.2024J2.3

**a. Tabla de capacidades**:

| Característica visual | Subtipo                | Variable cuantitativa | Variable ordinal | Variable categórica |
| --------------------- | ---------------------- | --------------------- | ---------------- | ------------------- |
| Color                 | Tonalidad (matiz, hue) | X                     | –                | +                   |
| Color                 | Intensidad             | +                     | +                | —                   |
| Color                 | Saturación             | —                     | —                | —                   |
| Forma                 | Orientación            | —                     | —                | —                   |
| Forma                 | Forma                  | X                     | —                | +                   |
| Forma                 | Cantidad               | +                     | +                | —                   |
| Posición espacial     | Posición 2D            | +                     | +                | —                   |
| Movimiento            | Desplazamiento         | —                     | —                | —                   |

**b. Justificación**:

Color–Tonalidad es adecuado para variables categóricas porque distingue clases sin implicar orden.

Color–Intensidad es adecuado para variables cuantitativas y ordinales porque permite una progresión perceptual clara (claro–oscuro), facilitando la representación de magnitud.
