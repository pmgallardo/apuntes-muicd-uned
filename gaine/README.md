# Gestión/almacenamiento de información no estructurada

En esta carpeta se incluyen recursos de apoyo educativo para la asignatura de Gestión y almacenamiento de información no estructurada (GAINE), con código 3111008-, del 2.° semestre del MUICD de la UNED.

Para más información, ver la [ficha oficial](https://www.uned.es/universidad/inicio/estudios/masteres/master-universitario-en-ingenieria-y-ciencia-de-datos/asignaturas.html?codAsignatura=3111008-&codTitulacion=311001).

## Evaluación

| Actividad evaluable | % nota final | % nota mínima |
| - | - | - |
| Trabajos prácticos (4) | 60 | 40 (de media) (4 sobre 10) |
| Exámen | 40 | 40 (4 sobre 10) |

### Trabajos prácticos

Se establecen 4 trabajos prácticos con un mismo peso cada uno.

| Actividad evaluadora | % nota final |
| Trabajo práctico 1 | 15 |
| Trabajo práctico 2 | 15 |
| Trabajo práctico 3 | 15 |
| Trabajo práctico 4 | 15 |

Se mantienen los enunciados y las notas de la convocatoria de junio para la de septiembre.

### Exámen

Puntuación mínima: 4 puntos.
Tipo exámen: 20 preguntas multiopción con 4 respuestas posibles
Puntuación correcta: +0.5
Puntuación incorrecta: -0.2 (-40%)

## Bibliografía

La bibliografía complementaria proporcionada en la ficha de la asignatura parece desactualizada, pues contiene títulos descatalogados y los más recientes son del 2019, cuando se inauguró el máster.

- Básica
  - Material original proporcionado por la asignatura.
- Complementaria
  - NoSQL
    - Gaurav Vaish. [Getting Started with NoSQL](https://learning.oreilly.com/library/view/getting-started-with/9781849694988/). Editorial: Packt Publishing, 2013.
    - Pramod J. Sadalage, Martin Fowler. [NoSQL Distilled: A Brief Guide to the Emerging World of Polyglot Persistence](httpss://learning.oreilly.com/library/view/nosql-distilled-a/9780133036138/ ). Addison-Wesley Profesional, 2012.
    - Xun Wu, Sudarshan Kadambi, Devram Kandhare, Aaron Ploetz. [Seven NoSQL databases in a Week](https://learning.oreilly.com/library/view/seven-nosql-databases/9781787288867/). Packt Publishing. 2018
    - Pethuru R., Deka, G. C. [A Deep Dive into NoSQL Databases: The Use Cases and Applications](httpss://www.oreilly.com/library/view/a-deep-dive/9780128137871/). Academic Press, 2018.
    - AKHTAR, Syed Muhammad Fahad. [Big Data Architect's Handbook: A Guide to Building Proficiency in Tools and Systems Used by Leading Big Data Experts](https://github.com/PacktPublishing/Big-Data-Architects-Handbook). Packt Publishing, 2018.
    - Daniel Ohene-Kwofie, E. J. Otoo, Gideon Nimako. O2-Tree: A Fast Memory Resident Index for NoSQL Data-Store. Conferencia: IEEE 15th International Conference on Computational Science and Engineering, 2012.
  - Redis
    - DAS, Vinoo. [Learning Redis](https://learning.oreilly.com/library/view/learning-redis/9781783980123/). Packt Publishing, 2015.
  - MongoDB
    - Kristina Chodorow, Shannon Bradshaw. [MongoDB: The definitive guide, 3rd edition](https://learning.oreilly.com/library/view/mongodb-the-definitive/9781491954454/). O'Reilly Media, Inc., 2019.
  - Cassandra DB
    - Jeff Carpenter, Eben Hewitt. Cassandra: The definitive guide. 3rd ed. O'Reilly Media, 2020.
    - Eben Hewitt, Jeff Carpenter. [Cassandra: The definitive guide](https://learning.oreilly.com/library/view/cassandra-the-definitive/9781491933657/), 2nd edition. O'Reilly Media, Inc. 2016.
  - Neo4j
    - WEBBER, Jim, VAN BRUGGEN, Rik. [Graph Databases: Neo4j Special Edition](https://neo4j.com/blog/graph-data-science/introducing-graph-databases-for-dummies/). Wiley, 2020.
    - VAN BRUGGEN, Rik, BATON, Jérôme. [Learning Neo4j 3.x - Second edition](https://learning.oreilly.com/library/view/learning-neo4j-3x/9781786466143/). Packt Publishing, 2017.
