# Programación en entorno de datos

En esta carpeta se incluyen recursos de apoyo educativo para la asignatura de Programación en entorno de datos del MUICD de la UNED.

Puedes ver la [ficha oficial de la asignatura](https://www.uned.es/universidad/inicio/estudios/masteres/master-universitario-en-ingenieria-y-ciencia-de-datos/asignaturas.html?codAsignatura=31110022&codTitulacion=311001&idContenido=1).

## Contenido

La carpeta se organiza en la siguiente estructura:

- **[`exam`](dist/pdf/exam/)**: Incluye soluciones y enunciados inspirados en exámenes anteriores.

## Evaluación

La asignatura se evalúa a partir de las siguientes actividades:

| Actividad | % |
| - | - |
| Actividad evaluable | 5 |
| Práctica obligatoria | 35 |
| Exámen | 60 |

### Actividades evaluables

Suponen un 5% de la asignatura.

Consiste en completar 3 actividades y subirlas al foro.

### Exámen

Puntuación máxima: 10.
Puntuación mínima para aprobar: 5 puntos (50%).

No se puede llevar calculadora ni material escrito.

Partes:

- Test  
  - 4 preguntas tipo test con 4 opciones posibles.  
  - Puntuación máxima: 4 puntos  
  - +1 preguntas correctas  
  - -0,3 preguntas incorrectas  
- Desarrollo  
  - Puntuación máxima: 6 puntos  
  - 2 preguntas de desarrollo, con una puntuación máxima de 3,5 y 2,5.  
  - Consiste en desarrollar un pequeño programa con Pandas para operar con un conjunto de datos  

En el curso 2021/2022 se estableció que era necesario aprobar ambas partes del exámen por separado, pero no se repitió en cursos posteriores.
