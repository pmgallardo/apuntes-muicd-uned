## Tipos de datos

### Slicing

Si se añade un tercer entero al slicing indica que se debe ir saltando los elementos según ese número, tras leer el primer elemento.

```python
seq = [7, 2, 4, 6, 3, 6, 8, 2]
print(seq[::2])
```

```text
[7, 4, 3, 8]
```

Si la tercera parte del split tiene un valor `-1`, se invierte el array en ese eje.

Si la tercera parte del split tiene un entero negativo, primero se invierte el array y después se van contando los números
